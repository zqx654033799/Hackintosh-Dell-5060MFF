# Hackintosh-Dell-5060MFF

## 平台配置
* 准系统：Dell Optiplex 5060 Micro
* CPU：Intel Core i5-8600T (Coffee Lake)
* GPU：Intel UHD Graphics 630
* 网卡：Apple BCM943224PCIEBT2
* 内存：Asgard 8GB DDR4 2666MHz * 2
* 存储：WD SN570 500G

## 特性
* 仿冒机型：Macmini8,1（SN已去除，需自行补充）
* 系统版本：Big Sur 11.6.6
* 显示正常
* 内置扬声器输出正常
* WiFi正常，支持Airdrop
* 蓝牙正常
* 睡眠正常
* USB端口正常
* 支持开机音效
